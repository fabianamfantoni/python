# PYTHON

*O Python é uma linguagem de programação extremamente versátil. É utilizada em diversas áreas, como ciência de dados, desenvolvimento web, automação e inteligência artificial.*

## Alguns pontos que fazem python ser uma linguagem mais "tranquila" de se aprender:

- Sintaxe simples: pouco verbosa, ao contrário de outras lingugens, mais próxima da linguagem humana.

- Bibliotecas poderosas: como NumPy, Pandas, Flask e TensorFlow.

- Multiplataforma: funciona em Windows, macOS, Linux e até dispositivos móveis.

- Comunidade ativa: milhões de desenvolvedores compartilham tutoriais e projetos.
